## Funcionalidades
### Usuário
- Cadastro de usuário.
- Login no sistema.

### Categoria
- Salvar uma categoria de tarefa.
- Editar uma categoria.
- Listar suas categorias.
- Excluir uma categoria.
- Alterar o status de uma categoria.

### Tarefa
- Criar uma tarefa.
- Editar tarefa.
- Excluir Tarefa.
- Listar e filtrar tarefas.
